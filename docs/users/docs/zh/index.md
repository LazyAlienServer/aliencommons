# AlienCommons 用户指南

AlienCommons 是一个以 Minecraft 为核心的社区，用于发布内容、讨论、发现作品和参与社区活动。

## 社区范围

帖子、文章、评论、个人资料和其他社区内容原则上应当与 Minecraft 有关。偶尔出现相邻话题或日常话题可以接受，例如硬件、软件、创作工具或生活日常，只要它们不扰乱社区秩序。

AlienCommons 对商业噪音保持严格态度。广告、垃圾信息、诈骗、刷屏推广、低价值引流内容，即使表面上和 Minecraft 有一点关联，也可能被移除。

## 禁止内容

以下内容不允许出现：

- 政治、意识形态或高度敏感的社会争议内容。
- 色情、露骨性内容或性化内容。
- 暴力、血腥、威胁或危险内容。
- 扰乱社区的宗教争议或传教内容。
- 仇恨、歧视、非人化表达，或针对身份群体的攻击。
- 违法活动、诈骗、人肉、盗号，或帮助他人造成伤害的指引。
- 人身攻击、骚扰、霸凌、恐吓或定向辱骂。
- 垃圾信息、无关推广、疑似机器刷屏，以及其他破坏社区生态的行为。

## 举报分类

举报内容或用户资料时，请选择最接近的问题：

- 偏离 Minecraft 主题或低质量内容。
- 垃圾广告、推广或刷屏。
- 人身攻击、骚扰或辱骂。
- 仇恨或歧视。
- 色情或性内容。
- 暴力、血腥或威胁。
- 政治、意识形态或社会争议内容。
- 宗教争议内容。
- 违法、诈骗或危险行为。
- 其他问题。

举报会帮助审核人员理解发生了什么。如果内容属于善意的相邻话题，或没有违反社区规则，举报也可能被驳回。
